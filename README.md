# 🎮 FriendTiers

Eine interaktive Tier-List-Website für dich und deine Freunde!

Inspiriert von MCTiers - erstelle deine eigene Ranking-Liste und teile sie mit deinen Freunden.

---

## ✨ Features

- ✅ **Drag & Drop Interface** - Einfach Items verschieben
- ✅ **6 Tier-Levels** - S, A, B, C, D, F (wie bei MCTiers!)
- ✅ **Automatisches Speichern** - Deine Liste wird im Browser gespeichert
- ✅ **Responsive Design** - Funktioniert auf PC, Tablet und Handy
- ✅ **Einfach zu bedienen** - Keine Anmeldung nötig

---

## 🚀 Wie benutzt man es?

### Online nutzen (einfach!)
Geh einfach auf die Website: **https://pek67.github.io/Friendtiers/**

### Lokal testen
1. Repository klonen:
```bash
git clone https://github.com/Pek67/Friendtiers.git
cd Friendtiers
```

2. `index.html` im Browser öffnen (oder einen lokalen Server starten)

---

## 📝 Schritt-für-Schritt Anleitung

1. **Namen eingeben** - Gib einen Namen (Freund, Creator, etc.) in das Textfeld ein
2. **Hinzufügen klicken** - Der Name erscheint in "Nicht eingeordnet"
3. **Verschieben** - Ziehe den Namen in die gewünschte Tier-Box
4. **Speichern** - Automatisch! Deine Liste wird gespeichert
5. **Löschen** - Klick auf das ✕-Symbol um einen Namen zu löschen

---
